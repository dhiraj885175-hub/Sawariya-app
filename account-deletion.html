🗑️ Sawariya — Account Delete Request

Agar aap apna Sawariya account aur usse juda data permanently delete karwana chahte hain, to neeche diye gaye kisi bhi tareeke se request bhej sakte hain.

Request bhejne ka tareeka:

  1. 1Apna registered email/phone number jo app mein use kiya tha, neeche diye gaye WhatsApp ya Email par bhejein
  2. 2Subject/message mein likhein: "Account Delete Request"
  3. 3Hum 7 working days ke andar aapka account aur personal data delete kar denge
💬 WhatsApp se Request Bhejein 📧 Email se Request Bhejein
Delete hone wala data:
Naam, email, phone number, saved address, account login jaankari.

Note: Kanooni/business record ke liye purane orders ka basic (non-personal) transaction history rakha ja sakta hai.

Sawal ho to upar diye gaye contact se hume sampark karein. Hum aapki request jald se jald process karenge.
